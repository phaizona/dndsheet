# dndsheet# 🐉 Fantasy Interactive D&D Character Sheet

A beautifully designed, interactive D&D 5e-style character sheet. This project brings a touch of fantasy to your digital gaming sessions with a custom parchment-inspired aesthetic and fully functional dynamic input fields.

## 🚀 How to Use

You can start using the character sheet in two ways:

1. **Online Version:** Simply visit [https://phaizona.github.io/dndsheet/](https://phaizona.github.io/dndsheet/) to use it directly in your browser.
2. **Local Version:** Clone or download this repository, then open `index.html` in your web browser to use it offline.

*Fill in your character details, upload a portrait, and click **Download as PNG** or **PDF** to save your sheet whenever you're ready.*

## 📜 Features

*   **Parchment Aesthetics:** Custom typography and UI elements that feel like a real RPG document.
*   **Dynamic Layout:** Responsive grid system that ensures all stats, skills, and notes stay perfectly aligned.
*   **Smart Textfields:** Text areas that automatically expand as you type—never run out of room for your backstory or equipment!
*   **Interactive Stats:** Automatic calculation of ability modifiers based on your core stats.
*   **Export Options:** Seamlessly save your character as a high-quality **PNG** or **PDF** file.
*   **Custom Checkboxes:** Elegant, hand-drawn style checkboxes for saving throws and death saves.
*   **Portrait Support:** Easy drag-and-drop or click-to-upload character portrait system.

## 🛠 Tech Stack

*   **HTML5, CSS3, JavaScript** (Vanilla)
*   **Libraries:** `html2canvas` (for image generation) & `jsPDF` (for PDF generation)
